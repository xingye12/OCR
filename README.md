## 使用方法：

下载源码后，将文件夹在Android studio中打开
在科大讯飞申请自己的应用，开通免费的离线语音包服务，将APP中的app keyid改成自己的app keyid即可使用科大讯飞离线语音
将软件打包apk下载到真机使用
首页只能点击“中英文拍照识别”（其他内容暂未完成开发）进入拍照界面（即主界面），所有的按钮均需要长按才可以生效对应的点击事件
